# Easy Podcast

## Tech stack

- **Frontend:** React 18 + Vite
- **Mobile:** Capacitor (to generate the Android APK)
- **Language:** JSX (all logic in a single file `src/App.jsx`)
- **Build:** `npx vite build` → outputs `www/` → `npx cap sync` → `cd android && ./gradlew assembleDebug`

## Folder structure

```
~/Desktop/easy-podcast/
├── src/
│   ├── App.jsx        ← All app logic and UI
│   └── main.jsx       ← React entry point
├── www/               ← Vite output (do not edit, regenerated on each build)
├── android/           ← Android project generated by Capacitor
├── index.html         ← Vite root HTML
├── vite.config.js     ← Build outDir: 'www'
├── capacitor.config.json
└── package.json
```

## Android SDK

- **Installed at:** `~/Library/Android/sdk`
- **Environment variables in `~/.zshrc`:**

```bash
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/platform-tools
export JAVA_HOME=$(/usr/libexec/java_home -v 21)
export PATH=$JAVA_HOME/bin:$PATH
```

- **Active Java:** OpenJDK 21 (Amazon Corretto)
- **Test device:** Pixel 8a (connected via USB with USB debugging enabled)

## Current app state

The app is a functional prototype of a podcast client with:

- 4 screens: Home, Search, Queue, Library
- Mini player and full-screen player
- Navigation, favorites, downloads, reorderable queue (drag & drop)
- Persistence via `localStorage`
- **Real audio** — HTML5 `<audio>` element with play/pause/seek/speed and auto-advance to next episode
- **No real API** — podcasts and episodes are hardcoded data in `App.jsx` (placeholder audio: SoundHelix MP3)

## Pending next steps
